# searching_algorithms_activity
A simple CSA search algorithms activity

### Tasks
1. Complete the linear search algorithm. Run and test
2. Run the "runTimedMethods" method using a list of size 15. Answer the questions in the Notebook
3. Change the size of the array. Try a variety of numbers (I went as high as 6000!). Write your observations

### Submission
Finish the code and submit in your GitHub repo. Complete the questions in the Notebook.
